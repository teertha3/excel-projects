## Sales Data Analysis using Excel

### Objective
To analyze sales data and generate insights using Excel formulas, pivot tables, and visualizations.

### Dataset Source
- Sales dataset obtained from a public GitHub repository:
  - https://github.com/shadsluiter/ExcelExamples
- Dataset used strictly for learning and analysis purposes.

### Excel Features Used
- Text to Columns for splitting full names into First Name and Last Name
- Formatting monetary values to dollars (USD)
- IF function to calculate sales commission based on predefined conditions
- SUMIF function to calculate conditional totals 
- Sorting and filtering sales records
- Pivot Table creation for summarized analysis (Sheet2)
- 3D Pie Chart for sales distribution visualization

### Description of Work
- Imported the sales dataset into Excel
- Split customer names into first and last names using Text to Columns
- Formatted monetary values for consistency
- Applied IF formulas to calculate commission amounts
- Used SUMIF to compute conditional totals
- Sorted and filtered data to analyze specific sales segments
- Created a pivot table in a separate sheet to summarize sales performance
- Visualized sales distribution using a 3D pie chart

### Key Insights
- Clear understanding of sales distribution across categories
- Identification of high-performing sales segments
- Simplified analysis using pivot tables and charts

### Key Learning Outcomes
- Practical experience with Excel data cleaning techniques
- Applying conditional and aggregation formulas
- Hands-on practice with pivot tables and data visualization
- End-to-end Excel data analysis workflow

### Tools Used
- Microsoft Excel
