## Payroll Calculation using Excel

### Objective
To calculate employee payroll based on hourly wages, weekly working hours, and overtime rules using Excel formulas.

### Dataset
- Payroll.xlsx  
- Manually structured payroll sheet containing employee details and weekly work hours.

### Excel Features Used
- Arithmetic formulas for salary calculation
- Overtime calculation logic (hours greater than 40)
- Weekly wage computation
- Monthly total salary calculation
- Date-based column organization
- Basic data formatting

### Description of Work
- Entered employee hourly wage and weekly working hours
- Calculated weekly salary using:
  - Regular pay = Hourly Wage Ã— Regular Hours
  - Overtime pay applied for hours exceeding 40
- Computed total pay for each week
- Aggregated weekly salaries to obtain total monthly pay (January Pay)

### Key Learning Outcomes
- Understanding real-world payroll calculation logic
- Applying Excel formulas for business calculations
- Organizing time-based payroll data
- Avoiding errors in salary computation

### Tools Used
- Microsoft Excel
