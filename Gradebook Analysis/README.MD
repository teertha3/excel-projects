## Employee Gradebook Analysis using Excel

### Objective
To analyze employee assessment performance by calculating scores, percentages, and visualizing results using Excel charts.

### Dataset
- Gradebook.xlsx  
- Contains individual assessment scores across multiple tests along with calculated percentages and summary statistics.

### Assessments Included
- Safety Test
- Company Philosophy Test
- Financial Skills Test
- Drug Test

### Excel Features Used
- Arithmetic formulas for score and percentage calculations
- Statistical functions: MAX, MIN, AVERAGE
- Logical formulas to determine employee status (Fire Employee: TRUE/FALSE)
- Conditional formatting to highlight low and high performance
- Column charts for visual comparison of assessment scores
- Structured tabular data organization

### Description of Work
- Entered assessment scores for each individual
- Converted raw scores into percentage values
- Calculated maximum, minimum, and average scores for each test
- Applied logical conditions to flag employees based on performance criteria
- Used conditional formatting to visually identify poor performance
- Created multiple column charts to compare performance of employees across assessments

### Key Insights
- Quick identification of low-performing individuals
- Visual comparison of performance across different tests
- Clear understanding of overall score distribution using charts
- Improved decision-making using logical flags and summaries

### Key Learning Outcomes
- Practical experience with Excel formulas and statistical functions
- Using conditional formatting for data interpretation
- Creating charts to visualize performance data
- Applying Excel for real-world evaluation and decision-making scenarios

### Tools Used
- Microsoft Excel
